## About
通过scrapy爬虫框架采集银行代码一览表, 并导出json文件

## Usage
```
# 导出银行代码json文件
python start.py

# 检查请求ip是否为代理ip
scrapy crawl myip

# 检查请求头参数
scrapy crawl my_headers

# 检查请求user agent
scrapy crawl my_user_agent
```
